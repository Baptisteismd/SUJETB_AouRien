Le programme associé à la SPEC_5 est GenerateurGraph.js.
Pour le lancer indépendamment du main, l'utilisateur doit s'assurer que le chemin du terminal mène au dossier SPEC_5 contenant le programme.
Pour activer le programme efficacement, il faut s'assurer qu'il existe des fichiers .gift dans le répertoire (fichiers constitués de questions d'examens).

Pour générer des fichiers .gift, il est possible de récupérer les 2 fichiers du dossier FonctionsSupplementaires (GenerateurExamens.js et ListTotQ.txt) et de les ajouter au répertoire SPEC_5.
Il faut ensuite lancer le programme GenerateurExamens.js et écrire la commande "node GenerateurExamens.js" dans le terminal pour exécuter le programme. L'utilisateur devra alors suivre les instructions du programme jusqu'à ce que le programme ait arrêté de s'exécuter.

Une fois le répertoire SPEC_5 constitué de fichiers .gift et le chemin du terminal correspondant à l'emplacement du dossier, il est possible de lancer le programme répondant à la SPEC_5 en ouvrant le fichier GenerateurGraph.js et en rentrant dans le terminal la commande suivante : node GenerateurGraph.js
L'utilisateur devra alors suivre les instructions du programme jusqu'à ce que le programme arrête de s'exécuter.

Les fichiers package.json et package-lock.json sont indissociables du programme, ils permettent notamment d'importer la bibliothèque de VegaLite.
