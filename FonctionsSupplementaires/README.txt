Ce dossier possède un fichier appelé GenerateurExamens.js.
C'est un programme fonctionnel, bien que hors projet, qui permet de faciliter les tests des différentes SPECs nécessitant une base de donnée de questions en format .gift.
Pour fonctionner, il utilise les données du fichier ListTotQ.txt.
Par défaut, exécuter GenerateurExamens.js va générer des fichiers .gift dans le répertoire où il se situe. Il faut donc le placer dans le dossier de la SPEC qui intéresse pour que les fichiers créés soient bien placés.

Pour lancer le programme, l'utilisateur doit s'assurer que le chemin du terminal mène au dossier FonctionsSupplementaires contenant le GenerateurExamens.js.
Il est ensuite possible de lancer le programme en écrivant la commande "node GenerateurExamens.js" dans le terminal pour exécuter le programme. L'utilisateur devra alors suivre les instructions du programme jusqu'à ce que celui-ci ait arrêté de s'exécuter.
